서비스 전면 개편

1. 서비스 명: 독독 (처음 이야기한 것과 동일함)
2. 주요 설명:

1)  하나의 독서 모임에 대해 누구나 접속 코드로 들어오면 거기서 독서 모임에 대한 내용을 남기고 기록할 수 있음. 그리고 그걸 갤러리 형태, 타임라인 형태로 볼 수 있으면 좋겠음
2)  독서 모임에 대한 내용을 남길 때는 모임 날짜, 책 검색하기, 발제자, 참여자, 발제문, 사진(첨부 할 수 있음), 글 남기기(후기나 하고 싶은 내용을 적을 수 있는 곳) 그리고 위와 관련해서 모두 crud 할 수 있어야 함
3)  프로필처럼 독서 모임이 하나 있어서 같이발제자는 이름으로 넣을건데 어떤 사람이 몇번씩 발제를 했는지 통계로 볼 수 있으면 좋겠음.
4)  ref의 이미지 참고해서 ui 해당 형태로 구성하고 싶음

- 평점 이미지는 평점 화면은 없지만 발제를 누가 몇번했는지 그래프처럼 보고 싶음
- 프로필 이미지는 없지만 프로필에는 독서 모임에 대한 프로필로 여기에 독서 모임에 대한 독서 모임 사진, 이름, 기록(독서모임 회수 등, 독서 모임에 입력한 내용이 자동으로 연동되어서 볼 수 있기를 바람), 참여한 사람, 발제자 통계 가 보이면 좋겠음

3. 주요 기능

- 회원가입 없이, 코드를 넣으면 접속할 수 있음.
- 첫 화면은 갤러리 형태로 리포브의 갤러리 화면을 참고하길 바람:
- 코드명은 독서모임으로 해줘. 독서모임이라고 접속 코드를 넣으면 누구나 접속할 수 있음
- 책에 대한 제목, 저자, 설명 등은 같이 검색할 때 보이면 좋겠음
- 친구들과 함께하는 전용 독서모임 웹앱으로 독서 모임 회수가 많아지면서 만드는 것임
