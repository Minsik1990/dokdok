# 시장 분석 & 경쟁 서비스

> 최종 업데이트: 2026-02-07 (v1.0)

---

## 1. 시장 현황

### 한국 독서 시장

- **성인 독서율**: 87.8% (10명 중 9명이 독서 경험 보유)
- **전자책 시장 CAGR**: 25.76% (2024~2032 예상)
- **MZ세대 트렌드**: "텍스트힙", 필사, "오독완(오늘의 독서 완료)"
- **핵심 기회**: 네이버/카카오 모두 독서 모임 전용 서비스가 없음

### 독서 모임 시장 특성

- 오프라인 독서 모임은 활발하나, 디지털 도구는 카카오톡/노션에 의존
- 모임 기록(발제문, 사진, 후기)이 메신저에 묻혀 유실됨
- 체계적인 모임 관리 + 아카이빙 수요 존재

## 2. 경쟁 서비스 분석

### 2.1 Repov (레포브) - 핵심 레퍼런스

- **개발사**: 밀리그램
- **포지셔닝**: "관점 기록 앱" (App Store 4.9점)
- **특징**:
  - 다양한 뷰 모드: 캘린더, 갤러리, 타임라인, 지도, 별점순
  - 둥글둥글한 카드형 UI, 미니 블로그형 인터페이스
  - 개인 기록 중심 (모임 기능은 약함)
- **Chaekdam 차별점**: 모임(그룹) 중심 + AI 독서 에이전트

### 2.2 트레바리

- **포지셔닝**: 프리미엄 독서 모임 서비스 (90억 투자 유치)
- **특징**:
  - 독후감 필수 제출
  - 체계적 모임 운영 (클럽장 + 파트너)
  - 월 구독료 5~8만원
- **Chaekdam 차별점**: 완전 무료, AI 기반 발제문 생성, 친구 모임 맞춤

### 2.3 StoryGraph

- **포지셔닝**: Goodreads 대안 (1인 개발 → 400만 유저)
- **특징**:
  - 감정/분위기 기반 책 추천
  - 독서 통계 & 시각화
  - 프리미엄 구독 모델
- **시사점**: 1인 개발로도 대규모 성장 가능

### 2.4 Letterboxd

- **포지셔닝**: 시네필 소셜 네트워크
- **특징**:
  - 문화 > 상업 (커뮤니티 우선)
  - 니치 시장 집중
  - 프리미엄 모델 수익화
- **시사점**: 문화 중심 접근이 장기 충성도를 만듦

### 2.5 기타 서비스

| 서비스      | 특징                      | 한계              |
| ----------- | ------------------------- | ----------------- |
| 북적북적    | 독서 기록 + 챌린지        | 모임 기능 약함    |
| 밀리의 서재 | 구독형 전자책 + 독서 모임 | 유료, 밀리 회원만 |
| 플라이북    | 큐레이션 중심             | 모임 기능 없음    |

## 3. Chaekdam 기회 요약

### 시장 공백

1. **무료** 독서 모임 관리 서비스 부재
2. 기존 도구(카톡/노션)의 아카이빙 한계
3. AI를 활용한 독서 모임 보조 서비스 전무
4. 감성적이고 따뜻한 UX의 독서 서비스 부족

### 타겟 유저

- 10~30명 규모의 오프라인 독서 모임
- 카카오톡으로 모임을 관리하는 MZ세대
- 모임 기록을 체계적으로 남기고 싶은 모임원

### 경쟁 우위

1. **완전 무료** (트레바리 대비)
2. **AI 독서 에이전트** (기존 서비스에 없는 기능)
3. **모임 중심** 설계 (Repov는 개인 기록 중심)
4. **따뜻한 감성** UI/UX (카카오/토스 스타일)
